# CPE-400-Project-Final

## Overview
This project analyzes network traffic generated by an Xbox One during gameplay, downloads, and streaming. The goal was to evaluate encryption and identify visible metadata.

## Tools Used
- Wireshark
- Python (Scapy, Pandas)

## Files
- analyze.py: Python script used to analyze packet captures
- xbox_gameplay_capature.zip: Gameplay capture
- xbox_streaming_capture.zip: Streaming capture
- (xbox_download_capture.zip is too large for GitHub)
- figures.zip: Figures used in report
  
## How to Run
1. Install dependencies:
   pip install scapy pandas

2. Run:
   python analyze.py

## Results Summary
- All traffic was encrypted (TLS)
- No plaintext sensitive data observed
- Metadata such as IP addresses and domains were visible
- Different usage scenarios showed distinct traffic patterns
